Olympic Games


By: Bill Liu, Luis Garcia, Tejas Babu










Project Description:

In this project we explore data from the Olympic games from 1896 to 2016. First we moved two large csv files into an sqlite database. Then, we created a flask application that would query data into different formats to then translate into creative visualizations. Each endpoint is in essence an ETL pipeline into D3 graphs. First the Data is queried with SQLAlchemy. Second, the data is transformed with Pandas into usefull information. Last, we convert the dataframes into json files that could be read by D3. The database looks at Olympic records from 1896 to 2016, that were originaly scrapped from www.sports-reference.com. To meet the requirement of using a javascript library not covered, we used Anime.js to generate the Olympic logo animation.
