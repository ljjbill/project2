Olympic Games


By: Bill Liu, Luis Garcia, Tejas Babu










Project Description:

Project Goal: To create a full stack application that pulled data from a database and visualized it using D3.js.

Data: In this project we explored data from the Olympic games spanning from 1896 to 2016. Originally scrapped from www.sports-reference.com.

Technology: Python, Sqlite database, SQLAlchemy, Pandas, HTML5, CSS, javascript, Anime.js, D3.js

  • First we created a Flask application that would server json endpoints to the front end
  • We then developed queries that would efficiently pull data from the database
  • Then we transformed the data into useful formats
  • Last, we used the enpoints to develop interactive visualizations of the data
